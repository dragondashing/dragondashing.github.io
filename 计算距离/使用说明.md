# 租房通勤距离计算器使用说明

## 打开方式

直接双击 `index.html`，用浏览器打开即可本地运行。

## 使用步骤

1. 在高德开放平台创建「Web 端 JS API」应用 Key。
2. 打开页面后，填写 `高德地图 Web JS API Key`。
3. 如果你的高德控制台开启了安全密钥，也填写 `Security JS Code`。
4. 填写城市，例如 `上海`、`北京`、`杭州`。公交/地铁换乘强烈建议填写城市。
5. 填写地址 A 和地址 B，点击「计算 A/B 距离」。
6. 填写候选租房地址 C，点击「计算 C 到 A/B 通勤」。

## 功能说明

- A/B 距离：展示 A 到 B 的直线距离，并给出驾车、步行、骑行、公交/地铁参考结果。
- C 到 A/B：展示 C 到 A、C 到 B 的直线距离，并列出不同交通方式的路线距离和预计时间。
- 页面会在地图上标记 A、B、C 三个位置。
- Key、密钥和城市只保存在当前浏览器的本地存储中。

## 注意事项

- 地址越完整，解析越准确，建议包含城市、区县、道路、门牌或大厦名称。
- 驾车、公交/地铁时间由地图服务估算，实际通勤时间会受到拥堵、等待、换乘和天气影响。
- 如果公交/地铁没有结果，请先确认城市已填写，并尝试把地址写得更具体。
